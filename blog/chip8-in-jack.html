Implementing CHIP-8 in Jack

2021-11-16

In this post, I will go over how I implemented a CHIP-8 emulator using Jack language (from the nand2tetris course). The emulator that it runs on is also made by me, so it is quite slow at the moment.

Given starting co-ordinates, we use the below function to draw a 64x32 screen somewhere with the screen. 


function void drawBox() {
    do Screen.drawLine(bx,    by,    bx+64,  by);      // top
    do Screen.drawLine(bx,    by,    bx,     by+32);   // left
    do Screen.drawLine(bx+64, by,    bx+64,  by+32);   // right
    do Screen.drawLine(bx,    by+32, bx+64,  by+32);   // bottom
    return;
}
